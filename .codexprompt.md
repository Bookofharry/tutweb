# Codex & Copilot Agent Guidelines

This file configures context for Codex, Copilot, and GPT models generating completions in this project workspace.

## Project Structure
A single-page static HTML blueprint website formatted for vertical and horizontal screen recording tutorials.

## Rules for Code Generation
1. **Preserve Single Column Layout**: Do not create complex multi-column layouts that break on narrow viewports or look small in social media crops.
2. **Massive Sizes Only**: Make sure titles and subtitles use fluid clamp styling to fill the horizontal canvas.
3. **Thick Borders**: Maintain the thick `4px` colored border scheme around `.content-card` tags.
4. **No UI Widgets**: Do not suggest or insert toggles, progress calculators, modal overlays, or navigation overlays. All sections must remain pre-expanded.
