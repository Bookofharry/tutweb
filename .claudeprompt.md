# Claude Agent Prompt Guidelines

This file configures Claude's context when editing files or writing code in this repository.

## Context
This project is a static linear scrolling website designed for content creators to screen-record social media educational tutorials.

## Rules for Claude
1. **Always Pre-Render Content**: Never collapse content behind interactive panels or tab buttons. The user scrolls down while filming; everything must show up instantly.
2. **Typography Constraints**: Do not reduce text sizes. Keep font scales set to large responsive fluid measurements:
   - Titles: `clamp(3rem, 7vw, 4.5rem)`
   - Details: `clamp(1.4rem, 3.5vw, 1.85rem)`
3. **Thick Border Accents**: Cards must use `4px` borders with custom properties (`--border-red`, `--border-green`, `--border-blue`, `--border-purple`, `--border-black`).
4. **No Frameworks or Dependencies**: Maintain raw HTML and CSS. Avoid importing Tailwind, React, or custom script packages unless explicitly asked.
